# shopnova-api

Project Overview: "ShopNova API"
A full-featured e-commerce backend with:

JWT + Passport authentication
Role-based authorization (user & admin)
Complete user & admin panels
OTP verification (email + phone via API, no 3rd-party OTP library)
BullMQ queues for async tasks
Cron scheduling
Email + in-app notifications
Prisma ORM
Clean, scalable project structure
All modern NestJS patterns (Guards, Interceptors, Pipes, Custom Decorators, etc.)

Tech Stack

NestJS 10+
TypeScript
Prisma + PostgreSQL
JWT + Passport
BullMQ + Redis
Nodemailer (for emails)
Class-validator + DTOs
Swagger documentation
ConfigService + dotenv
